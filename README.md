- [mobx-react源码](https://github.com/mobxjs/mobx-react)
- [mobx-react-lite源码](https://github.com/mobxjs/mobx-react-lite)

## 创建项目
```bash
pnpm create vite # 选则react
cd mobx-demo # 进入项目路径
pnpm i
pnpm add mobx mobx-react mobx-react-lite
pnpm dev
```
